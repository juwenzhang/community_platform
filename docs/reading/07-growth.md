## 1. 经典书单

### 必读（软件工程通识）

| 书名 | 作者 | 为什么读 |
|------|------|---------|
| [The Pragmatic Programmer](https://pragprog.com/titles/tpp20/) | David Thomas, Andrew Hunt | 程序员修炼之道，职业素养和编程哲学 |
| [Clean Code](https://www.oreilly.com/library/view/clean-code-a/9780136083238/) | Robert C. Martin | 代码整洁之道，命名/函数/注释/格式的艺术 |
| [Refactoring](https://martinfowler.com/books/refactoring.html) | Martin Fowler | 重构：改善既有代码的设计 |
| [Design Patterns](https://www.oreilly.com/library/view/design-patterns-elements/0201633612/) | GoF | 设计模式圣经（可先看 [Refactoring.Guru](https://refactoring.guru/) 可视化版） |

### 架构

| 书名 | 作者 | 为什么读 |
|------|------|---------|
| [Designing Data-Intensive Applications](https://dataintensive.net/) | Martin Kleppmann | DDIA，分布式系统必读，涵盖存储/复制/分区/事务/一致性 |
| [Building Microservices](https://samnewman.io/books/building_microservices_2nd_edition/) | Sam Newman | 微服务架构设计实战 |
| [Software Architecture: The Hard Parts](https://www.oreilly.com/library/view/software-architecture-the/9781492086888/) | Neal Ford | 架构取舍决策 |
| [A Philosophy of Software Design](https://web.stanford.edu/~ouster/cgi-bin/book.php) | John Ousterhout | 简洁软件设计哲学 |

### 系统

| 书名 | 作者 | 为什么读 |
|------|------|---------|
| [CSAPP（深入理解计算机系统）](https://csapp.cs.cmu.edu/) | Bryant & O'Hallaron | 从底层理解程序如何运行 |
| [Operating Systems: Three Easy Pieces](https://pages.cs.wisc.edu/~remzi/OSTEP/) | Remzi Arpaci-Dusseau | 操作系统入门，免费在线 |
| [The Linux Programming Interface](https://man7.org/tlpi/) | Michael Kerrisk | Linux 系统编程权威参考 |

### Rust 专项

| 书名 | 为什么读 |
|------|---------|
| [The Rust Programming Language](https://doc.rust-lang.org/book/) | 官方入门书，免费 |
| [Programming Rust, 2nd Edition](https://www.oreilly.com/library/view/programming-rust-2nd/9781492052586/) | O'Reilly 出品，更深入 |
| [Rust for Rustaceans](https://rust-for-rustaceans.com/) | 进阶 Rust，适合读完官方书后 |
| [Zero To Production In Rust](https://www.zero2prod.com/) | Rust Web 开发实战（Actix/Axum） |
